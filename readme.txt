== Description ==
The WebsiteSetup Business Companion plugin extends the WebsiteSetup Business theme. Adding a number of features to the admin panel that enhances the theme\'s customizability.

* Lead Text
With the Lead Text feature, authors can now add a short text below the title on their posts or pages.

* Sidebar Content
Authors can add content to their posts or pages\' sidebar that appears on top of the widgets. This gives the ability to create unique \"text widget-like\" content that is unique for every post or page.

* Custom Gutenberg Blocks
Gutenberg blocks specially made for the WebsiteSetup Business theme.

== Installation ==
1. In your admin panel, go to Plugins and click the Add New button.
2. Click Upload Plugin and Choose File, then select the plugin\'s .zip file. Click Install Now.
3. Click Activate to use your new plugin right away.

== Frequently Asked Questions ==
= Can I use this plugin with other themes? =

No. Installing this plugin with other themes will give you unexpected results since the plugin targets WebsiteSetup Business theme-specific functions only.

= Is this plugin required by WebsiteSetup Business theme? =

No, it is not required but it is recommended to maximize the theme\'s functionality and editing experience.
